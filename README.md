# Sistema de Estoque em Python

Projeto simples de gerenciamento de estoque feito em Python para praticar organização de código e arquitetura em camadas.

## Funcionalidades

- Adicionar produtos
- Listar produtos
- Remover produtos
- Ver resumo da compra

## Estrutura do Projeto

Sistema_estoque
│
├── main.py
│
├── models
│   └── produto.py
│
├── services
│   └── produtos_service.py
│
└── ui
    └── menu.py

## Como executar o projeto

1. Clone o repositório

git clone https://github.com/Heitorhj/sistema-estoque-python.git

2. Entre na pasta do projeto

cd sistema-estoque-python

3. Execute o programa

python main.py

## Tecnologias utilizadas

- Python
- Programação orientada a objetos
- Arquitetura em camadas
